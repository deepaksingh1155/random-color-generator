# random-color-generator
Random Color Generator is a responsive web application built using HTML, CSS, and JavaScript. It generates random colors instantly and displays HEX and RGB codes. Users can copy color values with one click. The project features an attractive UI, smooth design, and mobile-friendly layout for better user experience and frontend practice.
# 🎨 Random Color Generator
